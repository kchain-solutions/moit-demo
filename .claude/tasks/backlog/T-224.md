## Description

Configure `TWIN_IOTA_GAS_STATION_ENDPOINT` for both Alpha and Beta TWIN Nodes. The Gas Station service sponsors transactions so individual org wallets don't need direct funding.

Options:
1. Use IOTA Foundation's public testnet Gas Station (if available)
2. Deploy own Gas Station instance using `iotaledger/gas-station` (see gas-station-reference.mdx)

### Acceptance criteria

- Both nodes can execute identity and notarization operations without manual wallet funding
- Gas Station endpoint configured in Docker Compose env vars
- Transaction costs are covered by the Gas Station sponsor wallet
