## Description

Add visual indicators in the frontend to distinguish real vs simulated IOTA operations:

- Badge/icon next to DID showing "Real" or "Simulated"
- Ledger entries with real on-chain references show IOTA Explorer link
- `GET /api/node` returns adapter mode so frontend can conditionally render indicators

### Acceptance criteria

- Frontend shows adapter mode status
- Real DIDs display with Explorer links
- Real notarizations display with on-chain reference links
- Simulated mode shows no Explorer links (same as current)
