## Description

Full flow with `ADAPTER_MODE=in-process`: login, register org (real DID), upload document (real notarization), verify on IOTA Explorer, view real ledger entries.

### Acceptance criteria

- Login succeeds
- Org registration creates real `did:iota:testnet:0x...`
- Document upload triggers real notarization
- Ledger shows real on-chain references
- DID and notarization are verifiable on IOTA Explorer
- Round-trip for DID creation < 8 seconds
- Round-trip for notarization < 5 seconds
