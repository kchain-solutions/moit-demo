## Description

Choose the TWIN Adaptor deployment model for Vietnam government systems (eCoSys, VNACCS):

### Options (from TWIN Whitepaper Reference Architecture)

1. **External TWIN Adaptor:** Standalone component outside the TWIN Node. Maximum flexibility, requires separate deployment and networking.

2. **In-node TWIN Adaptor:** Implemented as a TWIN Data App or DS Connector App within the existing TWIN Node. Simplest deployment, but couples adaptor lifecycle to node.

3. **Hybrid TWIN Adaptor:** Some functionality outside (e.g., protocol translation), some inside (e.g., activity publishing). Balances flexibility and simplicity.

### Decision criteria

- eCoSys availability (likely manual upload fallback, see RV-01)
- VNACCS accessibility (likely webhook pattern, see RV-02)
- Vietnam Cybersecurity Law constraints (data locality, see RV-03)
- Deployment simplicity for MOIT stakeholders
