## Description

Vite/esbuild emits `Duplicate key "fontSize" in object literal` at `client/src/components/Payments.jsx:212`. The `<span>` style object declares `fontSize: 12` and then again `fontSize: l === 'UCR' || l === 'Invoice' ? 11 : 12` in the same literal — the first key is silently dropped.

Replace the two declarations with a single conditional `fontSize` (or merge into a single ternary) so the warning disappears and the intended size is the one actually applied. Verify with `npm run build` that no esbuild warnings remain.
