## Description

Deploy two independent TWIN Nodes locally using Docker Compose, mirroring the production two-country topology (Alpha = export, Beta = import).

### Architecture

```
docker-compose.twin.yml

  db-alpha (MySQL 8.4.7, port 3306)
  twin-bootstrap-alpha (one-shot, creates DID Alpha)
  twin-node-alpha (port 3000)

  db-beta (MySQL 8.4.7, port 3307)
  twin-bootstrap-beta (one-shot, creates DID Beta)
  twin-node-beta (port 3001)
```

Each node has:
- Its own MySQL instance and data volume
- Its own bootstrap (creates a separate on-chain DID identity)
- Its own twin_data volume for state persistence

### Steps

1. Create `docker-compose.twin.yml` in `moit-demo/` based on tutorials.101 structure
2. Create `.env.alpha` and `.env.beta` with:
   - Same base config (TWIN_NETWORK=testnet, connectors)
   - Different MySQL ports (3306 vs 3307)
   - Different TWIN_PORT (3000 vs 3001)
   - Different TWIN_NODE_NAME ("TWIN Alpha" vs "TWIN Beta")
   - Different volume mounts
3. Bootstrap both nodes (sequential, each creates its own DID on testnet)
4. Verify both nodes respond independently:
   - `GET http://localhost:3000/info` (Alpha)
   - `GET http://localhost:3001/info` (Beta)
   - Login on each returns independent JWT sessions

### Integration with demo server

The demo server's `ADAPTER_MODE` (T-209) routes requests:
- `demo-alpha (port 4000)` -> `twin-node-alpha (port 3000)`
- `demo-beta (port 4001)` -> `twin-node-beta (port 3001)`

New env vars for the demo server:
```
TWIN_NODE_URL=http://localhost:3000   # in demo-alpha
TWIN_NODE_URL=http://localhost:3001   # in demo-beta
```

### Acceptance criteria

- Two TWIN Nodes running on localhost:3000 and localhost:3001
- Each has its own DID identity (different `did:iota:testnet:0x...`)
- Login on Alpha does not authenticate on Beta (session isolation)
- Identity created on Alpha is not visible on Beta (data isolation)
- `docker compose -f docker-compose.twin.yml up` starts everything
- Document: env var reference and startup instructions
- Total resource footprint: < 2GB RAM, < 5GB disk
